# Story-Based Challenges

Instead of "write a loop", give a mission.

Example:

The Farmer's Harvest

A farmer collected fruits:

fruits = ["apple", "apple", "banana", "orange", "banana"]

Count how many of each fruit he collected.

Teaches: dictionaries, loops

Lost Robot

A robot starts at position 0.

Commands:

["LEFT", "LEFT", "RIGHT", "RIGHT", "RIGHT"]

Find the final position.

Teaches: loops, state

Dungeon Treasure

A dungeon contains:

[10, -5, 20, -3, 15]

Positive values are treasure, negatives are traps.

Calculate the player's final score.

Teaches: iteration and conditions

add questions like this with easy and medium difficulties
